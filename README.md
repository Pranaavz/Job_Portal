# Job Portal Web Application

## About
A Job Portal Web Application built using Python, Django, HTML, CSS, JavaScript, and SQLite3.

## Features
- Candidate Registration & Login
- Company Registration & Login
- Job Posting
- Job Search
- Job Application
- Admin Panel
- Secure Authentication

## Technologies Used
- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite3

## Installation

bash
git clone https://github.com/Pranaavz/Job_Portal.git
cd Job_Portal
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
